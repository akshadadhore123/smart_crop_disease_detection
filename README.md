# 🌾 Smart Crop Disease Detection & Treatment Recommendation System Using AI

## 📌 Project Overview
Smart Crop Disease Detection System is an AI-based web application that detects crop diseases from leaf images and provides treatment recommendations to farmers.  
The system helps farmers identify plant diseases early and take proper action to increase crop yield.

---

## 🚀 Features
- 📷 Upload crop leaf image
- 🤖 AI-based disease prediction (CNN Model)
- 💊 Treatment recommendation system
- 🌍 Farmer-friendly web interface
- 📊 Disease confidence percentage display
- 🗂 Organized backend & frontend structure

## 📊 How It Works
1. User uploads crop leaf image.
2. Image is sent to backend server.
3. AI model analyzes image.
4. Disease is predicted.
5. Treatment recommendation is displayed.
